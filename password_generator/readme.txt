🔐 Password Generator

A simple command-line Python tool for generating secure random passwords based on user-defined criteria.

This project helped me improve:

- function decomposition
- input validation patterns
- working with character sets
- building CLI tools in Python
- structuring small applications

✨ Features
- Generate multiple passwords at once
- Custom password length
- Choose character sets:
- Option to exclude ambiguous characters (il1Lo0O)
- Input validation for all user prompts

🧠 How it works
1. The user defines:
- number of passwords
- password length
- allowed character types
2. The program builds a character pool and generates random passwords using Python’s random.choice().
