Hangman (Word Guessing Game)

This is a simple implementation of the classic Hangman game written in Python.
The player tries to guess a hidden word by suggesting letters or the full word within a limited number of attempts.

This project helped me practice:

- working with strings and lists
- implementing game logic
- using loops and conditions effectively
- structuring code into functions
- handling user input and validation

🧠 Features
- Letter-by-letter guessing
- Full word guessing option
- racking of already used letters and words
- Visual hangman stages
- Input validation (only alphabetic characters allowed)

🎮 How the game works
1. A random word is selected from a predefined word list
2. The player tries to guess the word letter by letter or by entering the full word
3. Each incorrect guess reduces the number of attempts
4. The hangman figure is gradually drawn as mistakes increase
5. The game ends when the word is guessed or attempts run out

In future versions, I plan to enhance the game with:
