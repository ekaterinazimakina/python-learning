🔮 Magic 8 Ball — Fortune Teller

This is a simple console-based Python program inspired by the classic Magic 8 Ball toy.
The program answers user yes/no questions with randomized predictions.

This project helped me improve:

- function decomposition
- loops and control flow
- user interaction handling
- working with random module
- building simple interactive CLI applications

✨ Features
- Personalized greeting using user name
- Randomized fortune responses
- Continuous question loop until user decides to stop
- Input validation for yes/no responses

🎮 How it works
1. The program greets the user and asks for their name
2. The user can ask any yes/no question
3. The program returns a random answer from a predefined list
4. After each answer, the user can choose to continue or exit the game

🔮 Planned improvements

In future versions, I plan to enhance the project with:

- UI improvements: Tkinter-based graphical interface, visual “Magic Ball” animation
- Personality system: different “modes” for the ball (serious, funny, sarcastic tone of voice)
- Animation effects: simulated thinking delay using time.sleep(), “thinking animation” before showing the answer
